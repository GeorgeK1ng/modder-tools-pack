How to use it:

Via command prompt:

Open command prompt and set working directory same as path to this script
Type name of the script or just drop it over cmd
Drag and drop valid JSON file
You also can specify many files by repeating step 3. but remember to separate them by space.
Via dragging and dropping files as well as folders with JSONs over script icon.

Must be installed Python 2.7
